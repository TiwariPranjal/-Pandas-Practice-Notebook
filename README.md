# 🐼 Pandas Practice Notebook

## 📌 Project Overview
This project contains practice exercises and examples using the Pandas library in Python.  
The notebook focuses on data manipulation, cleaning, filtering, grouping, merging, and basic exploratory data analysis techniques.

It is designed for beginners who want to strengthen their data analysis skills using Pandas.
